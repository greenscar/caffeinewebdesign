HOW TO USE LINK ORGANIZER:

1) Extract the files into the same directory.
3) Edit and save the html template file. The file
	contains a 5-step customization procedure.
	follow those instructions accurately.
4) Optionally, edit the css file if you want to
	customize the colors and apperance of the
	link organizer
5) Upload the files to your web server. They
	should reside in the same directory.
6) In case you need help, contact paul@baggethun.net
