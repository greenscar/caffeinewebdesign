About This Web Application

This is a small web application which demonstrates some uses of the DEWEB package, which accompanies the book JSP, Servlets, and MySQL.

At present, this web application has the following features:

  • A controller servlet that intercepts all requests for .html pages
  • Various beans to model the state of the application
  • Several view JSPs that display information back to the user.
